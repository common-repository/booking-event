Plugin that makes a post 'bookable', using the custom fields.

== Description ==

This plugin allows you to create a bookable event using the custom fields. A post creator can make a post 'bookable' selecting the custom field 'bookable_event' (automatically created at the plugin activation) and setting a value for it, e.g. the 'yes' value. You can also limit the booking, for example at day of event. To do this setting custom field 'bookable_event' as data, for example, 2010-01-31	. 


The registered users can add/remove their participation to an event (represented by a post) by links at the bottom of the post.
This plugin provides also a widget displaying the list of the last 'bookable events' added.

Require: avatars plugin. you can download from http://www.sterling-adventures.co.uk/blog/2008/03/01/avatars-plugin/

= How to use = 

Create a new post and add a custom field 'bookable_event' and setting a value for it, e.g. the 'yes' value.

On avatar admin setting set size as 60px.

== Installation ==

1. Upload the 'bookable_events' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optionally add 'Last bookable events' widget to your sidebar

== Internationalization ==

Support for: English (default), Italian.
